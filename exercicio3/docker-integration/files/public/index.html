Paths and endpoints

GET

  • / (index)
  • /client (Returns a list of all registered clients)
  • /client/{id} (Returns a client with the specified Id)
  • /product Returns a list of all registered products)
  • /product/{id} (Returns a product with the specified Id)
  • /sale (Returns a list of all registered sales)
  • /sale/{id} (Returns a sale with the specified Id)

POST

  • /client (Creates a new client register with the information in the body)
  • /product (Creates a new product register with the information in the body)
  • /sale (Creates a new sale register with the information in the body)

PUT

  • /client/{id} (Updates the specified client with the information in the body)
  • /product/{id} (Updates the specified product with the information in the body)
  • /sale/{id} (Updates the specified sale with the information in the body)

DELETE

  • /client/{id} (Deletes a client with the specified Id)
  • /product/{id} (Deletes a product with the specified Id)
  • /sale/{id} (Deletes a sale with the specified Id)
